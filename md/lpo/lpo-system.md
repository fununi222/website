## 目的
ホームの「System Optimization Dashboard」は、`index.html` からこのファイルを `fetch` して更新されます。  
そのため、ホームのレーダーチャートを変更したい場合は、まずこのファイルのスコア定義を編集します。

---

## 更新対象
- ホームの3本バー
  `技術適応力` / `プロジェクト推進力` / `課題解決・改善提案`
- ホームのレーダーチャート10項目
  `技術適応力` / `プロジェクト推進力` / `チーム貢献・SL力` / `課題解決・改善提案` / `ドメイン知識` / `IaC・構成管理` / `監視・可観測性` / `開発力(Python/Bash)` / `生成AI・LLM活用` / `AIエージェント運用`

---

## 更新手順
1. このファイル内の「スキル・ポートフォリオ」の数値を編集します。
2. 数値は `- **項目名**: 80` の形式を維持します。
3. 項目名は `index.html` 側の読み取りキーと一致している必要があります。
4. 保存後にホームを再読み込みすると、レーダーチャートとバーに反映されます。

---

## 注意点
- 項目名を変えるとホーム側で値を拾えなくなります。
- 数値の直後に補足を書くのは問題ありません。
  例: `- **技術適応力**: 90 (+5)`
- ホームの3本バーは次の3項目だけを見ています。
  `技術適応力` / `プロジェクト推進力` / `課題解決・改善提案`
- レーダーチャートは10項目を固定順で参照しています。

## 読み取りルール
- `index.html` では正規表現で `- **項目名**: 数値` を抽出しています。
- そのため、箇条書きの先頭 `-`、太字の項目名、コロンの後ろの数値は残してください。
- 数値は整数で書くのが安全です。
- ホーム表示に不要な本文は自由に書けますが、スコア行の書式は崩さないでください。
