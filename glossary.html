Glossary

ブログ内で出てくる専門用語や略語を、記事を読み進めやすい粒度で整理したページです。

// 読みながら迷いやすい単語をまとめた簡易リファレンス

使い方

記事内のリンク付き用語を押すと、このページ内の該当用語まで直接ジャンプします。今後も用語を追加して育てていける構成です。

OpenClaw

OpenClaw

ブラウザ操作、シェル実行、検索などを組み合わせてタスクを進める AI エージェント実行環境です。

このブログでは、作業を分解して進める「実務向けの操作レイヤー」として扱っています。

AI Agent

AIエージェント

文章を返すだけでなく、指示に応じてツールを使いながら実際の作業まで進める AI のことです。

Orchestrator

オーケストレーター

複数の処理やツールの流れを調整し、どの順番で何を実行するかを管理する役割です。

記事内では、OpenClaw が各専門スキルへ仕事を振り分ける司令塔として登場します。

Wrapper

ラッパー

複雑な処理を直接見せず、呼び出しやすい形に包んで使えるようにする仕組みです。

PoC

PoC

Proof of Concept の略で、「その案や技術が本当に成立するか」を小さく試す検証のことです。

OAuth

OAuth

外部サービスに対して、パスワードを直接渡さずにアクセス権を認可するための仕組みです。

Google Calendar API

Google Calendar API

Google カレンダーの予定取得や作成を、プログラムから行うための公式 API です。

localhost

localhost

今そのプログラムを動かしている自分自身の端末を指す名前です。ローカル開発や認証の戻り先によく使われます。

VPS

VPS

Virtual Private Server の略で、クラウド上で使える仮想サーバーです。24時間動かしやすい一方で、認証や通信制約に注意が必要です。

Playwright

Playwright

ブラウザを自動操作して、ログインや画面遷移のテスト、自動化を行うためのツールです。

vSphere

vSphere

VMware が提供する仮想化基盤の総称で、仮想マシンやホストをまとめて管理できます。

Rubrik

Rubrik

バックアップやリストア、データ保護を効率化するエンタープライズ向けの製品群です。

CBT

CBT

Changed Block Tracking の略で、前回バックアップ以降に変化したブロックだけを追跡する仕組みです。

VAST Data

VAST Data

大規模データ処理向けのストレージ基盤を提供する製品・ベンダー群です。容量監視や保持設計の文脈で登場します。

BCP

BCP

Business Continuity Plan の略で、障害や災害時にも業務を続けるための継続計画です。

OSS

OSS

Open Source Software の略で、ソースコードが公開され、利用や改変がしやすいソフトウェアを指します。

Prometheus

Prometheus

メトリクス収集と監視に特化した OSS です。サーバーやアプリの状態を数値で追う用途に向いています。

Grafana

Grafana

Prometheus などのデータをグラフやダッシュボードで可視化する OSS です。

n8n

n8n

複数サービスをつなぐワークフロー自動化ツールです。ノーコード寄りですが、必要に応じて細かい制御もできます。
