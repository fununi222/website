Win11 4K 110% スケールと
FancyZones

"広大な作業領域と視認性のトレードオフ。110%という絶妙なカスタムスケーリングへの到達。"

1. なぜ「110%」なのか

Windows 11 の 4K モニター環境において、標準の 100% は文字が小さすぎて目が疲れ、125% は作業領域が削られすぎると感じることがあります。

設定の「カスタムスケーリング」から直接 **110** を入力することで、ドットバイドットのキレを維持しつつ、実用的な視認性を確保する「黄金比」に到達しました。

grid_view 2. FancyZones による完璧な配置

PowerToys の中で最も重要な機能である **FancyZones**。これを使い、ブラウザ、ターミナル、エディタをミリ単位で固定配置しています。

Main Zone (Middle)

VS Code or Main Browser. width: 60%

Side Zones

Slack, Terminal, or Reference. width: 20% each

3. カスタムスケーリングの注意点

カスタムスケーリングを使用すると、一部の古いアプリで表示がボケる（DPIスケーリングの非対応）ことがありますが、最新のブラウザや開発ツールであれば全く問題ありません。設定反映にはサインアウトが必須です。
