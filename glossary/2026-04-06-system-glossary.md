# System Glossary

ブログ内で登場する専門用語や略語を、記事を読み進めやすい粒度で整理したページです。
以下の表は、各ヘッダーをクリックしてソートしたり、キーワードで検索することができます。

| 用語・技術名 | カテゴリ | 概要・詳細 |
|---|---|---|
| OpenClaw | AI / Dev | ブラウザ操作、シェル実行、検索などを組み合わせてタスクを進めるAIエージェント実行環境。実務作業向けの操作レイヤーとして活用。 |
| AIエージェント | AI | 文章を返すだけでなく、指示に応じてツールを使いながら自律的に作業を進行するAIモデル。 |
| オーケストレーター | AI / Infra | 複数の処理やツールの流れを調整し、どの順番で何を実行するかを管理する統合機能。 |
| ラッパー (Wrapper) | Dev | 複雑な機能や処理を直接見せず、シンプルかつ安全に呼び出しやすい形に包んで提供する仕組み。 |
| PoC | Dev / Infra | Proof of Conceptの略。その設計アイデアや技術が「本当に業務で成立するか」を小さく試すシステム検証プロセスのこと。 |
| OAuth | Security | 外部サービスに対して、パスワードを直接渡さずにアクセス権限のみを認可するための標準的な仕組み。 |
| Google Calendar API | Infra | Googleカレンダーの予定取得や作成を、外部プログラムから自動操作・連携するための公式インタフェース。 |
| vSphere | VMware / Infra | VMwareが提供する仮想化基盤の総称。仮想マシン、ストレージ、ネットワークを統括して管理するプラットフォーム。 |
| CBT | VMware / Infra | Changed Block Trackingの略。バックアップ製品が「前回以降に変化したブロックだけ」を特定し高速にバックアップするためのトラッキング機構。 |
| LPO | Analytics | Landing Page Optimizationの略。訪問者のコンバージョン率を最大化するために、ページ設計や文言のABテスト・分析を行うプロセス。 |
| Terraform | Infra | サーバーやネットワークなどのインフラ構築をプログラムの形で記述・管理（IaC）するOSSツール。 |
| Ansible | Infra | サーバーの設定作業やアプリケーションのデプロイを自動化するための構成管理ツール。 |
| ハルシネーション | AI | AIが、事実に基づかないもっともらしい嘘を出力してしまう現象のこと。 |
| IaC | Infra | Infrastructure as Code の略。サーバー構成などを手動ではなくコードで定義・構築・管理する手法。 |
| 自己回帰モデル | AI | 直前までの入力データ（文脈）をもとに、次に続く確率が最も高いデータ（単語など）を順番に予測して生成出力するAIの言語モデル形式。 |
| API | Dev / Infra | Application Programming Interface の略。ソフトウェア同士が情報をやり取りするための連絡窓口のようなもの。 |
| VPS | Infra | Virtual Private Server の略。物理サーバーの中で仮想的に専用サーバー同等の環境を構築して提供されるホスティングサービス。 |
| アルゴリズム | Dev | 問題を解決したり、目的を達成するための計算や処理の具体的な手順。 |
| レスポンシブ | Dev | Webサイトのデザインが、パソコンやスマートフォンなど閲覧する画面サイズ（幅）に合わせて自動的に最適化される手法。 |
| ポイ活 | Finance | ポイント活動の略。クレジットカードや各種決済サービスを工夫して組み合わせ、付与されるポイント還元を最大化する活動。 |
| 修行 | Finance | 特定のクレジットカードで「年間○万円利用すると永年年会費無料」「ポイントボーナス」といった特典を得るために、決済額を積み上げる行為の俗称。 |
| カスタムスケーリング | OS / Infra | Windows等において、デフォルトの選択肢（100%, 125%など）に縛られず、ユーザー定義の任意の倍率（例: 110%）で画面を拡大表示する設定機能。 |
| トラブルシュート | Process | システム障害や不具合が発生した際に、その原因を特定し解決・復旧を図る一連の技術的プロセス。 |
| プロジェクトマネジメント | Process | プロジェクトの目的を達成するため、タスクの進行、リソース（人員・予算）、スケジュールなどを計画・管理すること。 |
| Prometheus | Infra / Obs | 時系列データに基づいた監視・アラート通知を行うOSS。多次元のデータモデルと強力なクエリ言語(PromQL)が特徴。 |
| Grafana | Infra / Obs | PrometheusやDB等のデータを美しく可視化するダッシュボードツール。 |
| Zabbix | Infra / Obs | ネットワーク、サーバー、クラウドサービスなどを統合的に監視するエンタープライズ向けOSS。 |
| n8n | AI / Dev | 「フェアコード」ライセンスのワークフロー自動化ツール。ノードを繋いで複雑な自動化処理をGUIで構築可能。 |
| Ollama | AI | ローカル環境でLLM（Llama 3等）を簡単に実行・管理するためのツール。 |
| LLM | AI | Large Language Model（大規模言語モデル）の略。膨大なテキストデータから学習し、人間のような自然な文章を生成するAI。 |
| JWST | Space / Ops | ジェイムズ・ウェッブ宇宙望遠鏡。初期宇宙の銀河形成や星形成プロセスの観測において、従来の理論を覆すデータを次々と「デバッグ出力」している。 |
| CMB | Space / Physics | 宇宙マイクロ波背景放射。宇宙誕生直後の熱い状態から発せられた名残の光。宇宙というシステムの「初期化ログ」に近い情報を持つ。 |
| 標準模型 | Physics | 素粒子物理学の基本的な枠組み。宇宙の最小単位がどう振る舞うかを規定する「基本ソースコード」群。 |
| GPU | Infra / AI | Graphics Processing Unit。画像処理や行列演算に特化したプロセッサで、現代のAI学習・推論における中心的なハードウェア。 |
| VRAM | Infra / AI | Video RAM。GPU専用のメモリ空間。大型のLLMを動かすには、この容量が最大のボトルネックとなる。 |
| UI | Dev | User Interface（ユーザーインターフェース）。サービスと利用者の接点となる画面表示や操作感の総称。 |
| フロントエンド | Dev | Webサイトやアプリにおいて、ユーザーが直接目にし、操作する部分（HTML/CSS/JS等）。 |
| HTML | Dev | HyperText Markup Language。Webページの構造を定義するための標準的なマークアップ言語。 |
| CSS | Dev | Cascading Style Sheets。Webページのレイアウトや装飾（色、フォント、配置など）を指定するためのスタイルシート言語。 |
| 100万円修行 | Finance | 三井住友カード等において、年間100万円の利用を達成することで年会費永年無料やボーナスポイント獲得を目指すポイ活の通称。 |
| キャッシュレス決済 | Finance / Ops | 現金（紙幣・硬貨）を使用せず、クレジットカード、電子マネー、QRコード等で支払いを行う決済方式の総称。 |
| Olive | Finance | 三井住友フィナンシャルグループが提供する、銀行口座、カード決済、証券、保険などを1つのアプリで管理できるモバイル総合金融サービス。 |
| Mastercard | Finance | 世界最大級のクレジットカード決済ネットワーク。多くのプリペイドカードへのチャージにおいて、Visaよりも制限が少ない傾向がある。 |
| Visa | Finance | 世界最大のクレジットカード決済ネットワーク。高い加盟店網を誇る。 |
| PowerShell | Dev / Ops | Microsoftが開発した、コマンドラインシェルおよびスクリプト言語。Windows管理やインフラ構築の自動化（Infrastructure as Code）に広く利用される。 |
| Jenkins | Dev / Ops | オープンソースの自動化サーバー。ビルド、テスト、デプロイなど、ソフトウェア開発に関わる一連のプロセスを自動化するための中心的なツール（CI/CD）。 |
| スナップショット | Infra / Ops | ある時点でのシステムやデータの状態を丸ごと保存した記録。VMware等の仮想化基盤では、更新前のバックアップや切り戻し用に頻繁に利用される。 |
| GitHub Actions | Dev / Ops | GitHub上でCI/CD（継続的インテグレーション/デリバリー）を実現するための自動化プラットフォーム。 |
| Cron | Ops | Unix系OSにおいて、スクリプトやコマンドを特定の時刻や間隔で自動実行するためのスケジュール管理システム。 |
| PowerToys | Windows / Ops | Microsoftが開発・提供する、パワーユーザー向けのオープンソース・システムユーティリティ群。ウィンドウ配置の最適化やファイル名の一括変更など、標準機能にはない高度なカスタマイズが可能。 |
| FancyZones | Windows / Ops | PowerToysに含まれるウィンドウ管理ツール。画面を任意のレイアウト（ゾーン）に分割し、ウィンドウをドラッグするだけで素早く配置できる機能。 |
| DNS | Network / Ops | Domain Name System（ドメイン名システム）。「google.com」のような人間が理解しやすい名前（ドメイン名）を、コンピュータが理解できるIPアドレスに変換する仕組み。 |
| NAT | Network / Ops | Network Address Translation。プライベートIPアドレスとグローバルIPアドレスを相互に変換する技術。 |
| パケットフィルター | Network / Ops | ネットワークの外部（インターネット）と内部（サーバー）の境界で、特定のルールに基づいて通信を通したり遮断したりする機能。ファイアウォールの一種。 |
| UL (Ultra Light) | Other / Hiking | ハイキングにおいて、装備の軽量化を突き詰めるスタイル。ベースウェイトを極限まで下げることで、より長く、速く、快適に歩くことを目的とする。 |
| ALTRA | Other / Hiking | 「ゼロドロップ」と「フットシェイプ」が特徴の米国のシューズブランド。自然な歩行姿勢を維持できるため、ULハイカーから絶大な支持を得ている。 |
