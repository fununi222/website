# System Glossary

ブログ内で登場する専門用語や略語を、記事を読み進めやすい粒度で整理したページです。
以下の表は、各ヘッダーをクリックしてソートしたり、キーワードで検索することができます。

| 用語・技術名 | カテゴリ | 概要・詳細 |
|---|---|---|
| OpenClaw | AI / Dev | ブラウザ操作、シェル実行、検索などを組み合わせてタスクを進めるAIエージェント実行環境。実務作業向けの操作レイヤーとして活用。 |
| AIエージェント | AI | 文章を返すだけでなく、指示に応じてツールを使いながら自律的に作業を進行するAIモデル。 |
| オーケストレーター | AI / Infra | 複数の処理やツールの流れを調整し、どの順番で何を実行するかを管理する統合機能。 |
| ラッパー (Wrapper) | Dev | 複雑な機能や処理を直接見せず、シンプルかつ安全に呼び出しやすい形に包んで提供する仕組み。 |
| PoC | Dev / Infra | Proof of Conceptの略。その設計アイデアや技術が「本当に業務で成立するか」を小さく試すシステム検証プロセスのこと。 |
| OAuth | Security | 外部サービスに対して、パスワードを直接渡さずにアクセス権限のみを認可するための標準的な仕組み。 |
| Google Calendar API | Infra | Googleカレンダーの予定取得や作成を、外部プログラムから自動操作・連携するための公式インタフェース。 |
| vSphere | VMware / Infra | VMwareが提供する仮想化基盤の総称。仮想マシン、ストレージ、ネットワークを統括して管理するプラットフォーム。 |
| CBT | VMware / Infra | Changed Block Trackingの略。バックアップ製品が「前回以降に変化したブロックだけ」を特定し高速にバックアップするためのトラッキング機構。 |
| LPO | Analytics | Landing Page Optimizationの略。訪問者のコンバージョン率を最大化するために、ページ設計や文言のABテスト・分析を行うプロセス。 |
| Terraform | Infra | サーバーやネットワークなどのインフラ構築をプログラムの形で記述・管理（IaC）するOSSツール。 |
| Ansible | Infra | サーバーの設定作業やアプリケーションのデプロイを自動化するための構成管理ツール。 |
| ハルシネーション | AI | AIが、事実に基づかないもっともらしい嘘を出力してしまう現象のこと。 |
| IaC | Infra | Infrastructure as Code の略。サーバー構成などを手動ではなくコードで定義・構築・管理する手法。 |
| 自己回帰モデル | AI | 直前までの入力データ（文脈）をもとに、次に続く確率が最も高いデータ（単語など）を順番に予測して生成出力するAIの言語モデル形式。 |
| API | Dev / Infra | Application Programming Interface の略。ソフトウェア同士が情報をやり取りするための連絡窓口のようなもの。 |
| VPS | Infra | Virtual Private Server の略。物理サーバーの中で仮想的に専用サーバー同等の環境を構築して提供されるホスティングサービス。 |
| アルゴリズム | Dev | 問題を解決したり、目的を達成するための計算や処理の具体的な手順。 |
| レスポンシブ | Dev | Webサイトのデザインが、パソコンやスマートフォンなど閲覧する画面サイズ（幅）に合わせて自動的に最適化される手法。 |
| ポイ活 | Finance | ポイント活動の略。クレジットカードや各種決済サービスを工夫して組み合わせ、付与されるポイント還元を最大化する活動。 |
| 修行 | Finance | 特定のクレジットカードで「年間○万円利用すると永年年会費無料」「ポイントボーナス」といった特典を得るために、決済額を積み上げる行為の俗称。 |
| カスタムスケーリング | OS / Infra | Windows等において、デフォルトの選択肢（100%, 125%など）に縛られず、ユーザー定義の任意の倍率（例: 110%）で画面を拡大表示する設定機能。 |
| トラブルシュート | Process | システム障害や不具合が発生した際に、その原因を特定し解決・復旧を図る一連の技術的プロセス。 |
| プロジェクトマネジメント | Process | プロジェクトの目的を達成するため、タスクの進行、リソース（人員・予算）、スケジュールなどを計画・管理すること。 |
